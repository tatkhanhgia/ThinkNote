## Prisma v7 Adaptation
Phase spec assumed Prisma v5/v6 patterns. Prisma v7 has breaking changes:
1. `url` removed from `schema.prisma` datasource -> moved to `prisma.config.ts`
2. `PrismaClient` requires a driver adapter (`PrismaPg`) instead of inline connection string
3. `prisma.config.ts` must be created (auto-generated by `prisma init`)
